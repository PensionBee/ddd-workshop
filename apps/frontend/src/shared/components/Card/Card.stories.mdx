# Card

## Props

There are 3 types of card: `Standard Card`, `Button Card` and `Link Card`. The props table below outlines the specific props for each.

- Passing `Button Card` props results in a `<button>` element being rendered in the DOM.
- Passing `Link Card` props results in an `<a>` element being rendered in the DOM.
- Passing neither `Button Card` props or `Link Card` props results in a `<div>` element being rendered in the DOM.
