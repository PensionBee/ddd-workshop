# Button

## Props

There are 2 types of button: `Standard Button` and `Link Button`. The props table below outlines the specific props for each.

- Passing `Standard Button` props results in a `<button>` element being rendered in the DOM.
- Passing `Link Button` props results in an `<a>` element being rendered in the DOM.
